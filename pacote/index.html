Pacotes

   
     

Na linguagem Java existe uma maneira simples e direta de se organizar     arquivos, além de possibilitar que se utilize mais de uma classe de mesmo     nome e no mesmo projeto. Um pacote (package) é um namespace     usado para organizar um conjunto de interfaces e classes relacionadas.     Podemos, por analogia, pensar nos pacotes como pastas que contém     classes que trabalham em conjunto.
